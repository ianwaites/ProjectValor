Valor

Our Mission

Project Valor is a tool built for the veterans of Washington State and can act as a resource for the Washington State Department of Veteran's Affairs. Valor can be used to locate the nearest facilities to the veteran's location and displays average wait times of each facility. We did this with publicly accessible information from the VA.

This tool can be used to make sure veterans, no matter where they live, have the information to choose the location that best serves their needs. Whether it is the nearest location or the lowest wait time.

Valor can show where facilities are understaffed, where the size of the facility is not enough for the needs of veterans, and where new facilities should be built to be able to positively impact the most veterans in need.
